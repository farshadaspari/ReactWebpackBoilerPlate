Farshad's React, Redux and Webpack boilerplage project.
Created by Farshad Aspari, October 2016

Installation:
-------------------------------------------------------------
NA


Use:
-------------------------------------------------------------
To run in dev mode:
	> npm run dev


Notes:
--------------------------------------------------------------
1. Webpack needs to be installed globaly:
	> npm install -g webpack

2. Nodemon needs to be installed gloably:
	> npm install -g nodemon
